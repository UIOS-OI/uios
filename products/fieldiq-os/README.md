# FieldIQ OS

FieldIQ OS is a product inside the UIOS platform, not the name of the umbrella repository.

It is the vendor-neutral enterprise AI operating system experience built on UIOS capabilities such as model routing, agent orchestration, memory, knowledge, workflows, plugins, MCP connectivity, analytics and security.

FieldIQ OS uses AegisAgent as UIOS’s built-in security system. Agent runs, model calls, plugin actions, MCP tools, workflows and sensitive data flows should pass through the standard AegisAgent policy and audit boundary.

## Relationship to UIOS

```text
UIOS
└── FieldIQ OS
```

FieldIQ may eventually include its own dashboard, SDK experience, workflow builder and enterprise controls, while sharing UIOS core services and plugin contracts.
